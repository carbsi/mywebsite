Python

Python on korkean tason ohjelmointikieli, joka tunnetaan helppokäyttöisyydestään ja laajasta kirjasto- ja yhteisötuesta.

Pythonissa olen hankkinut kokemusta yli 6 vuoden ajan, säännöllisesti ja epäsäännöllisesti. Sain arvosanan 5/5 Jyväskylän ammattikorkeakoulun (JAMK) Python scripting -kurssilta vuonna 2018. Harjoitustyönä olen kehittänyt klassista Pong-peliä Pythonilla ja luonut skriptejä omiin projekteihini, kuten tietojen keräämiseen ja analysointiin. Työssäni olen automatisoinut salasanojen generointiprosessin Pythonilla, parantaen tehokkuutta ja vähentäen manuaalisen työn tarvetta. Lisäksi olen suorittanut useita Pythonin kursseja, jotka ovat syventäneet ymmärrystäni ohjelmointikielestä ja sen soveltamisesta erilaisissa tehtävissä.

JavaScript

JavaScript on keskeinen verkkosivustojen kehityksessä, sillä se mahdollistaa dynaamiset ja interaktiiviset käyttäjäkokemukset.


function helloWorld() {
    console.log("Hello, World!");
}

Java

Java on laajasti käytetty ohjelmointikieli, joka soveltuu erityisesti suurten ja monimutkaisten järjestelmien kehittämiseen.


public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

C++

C++ on tehokas ohjelmointikieli, joka tarjoaa matalan tason hallinnan ja on suosittu pelien ja suorituskykyisten sovellusten kehityksessä.


#include 

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
